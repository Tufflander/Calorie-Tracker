How the AI will plug in (read me)

  1. Create a free account on Vercel or Netlify. Deploy this file as a static site.
  2. Add a serverless function at /api/insights that calls OpenAI with your data points to generate coaching tips.
  3. Store data long‑term in a free tier database such as Supabase. Replace localStorage with API calls.

Below, the “AI Insights” button tries to call /api/insights. If that route does not exist yet, you will see a friendly message.

Example Node serverless function (Vercel/Netlify) — pseudo‑code 
// file: api/insights.js
export default async function handler(req, res){
  const { entries } = await req.json(); // [{date, weight, kcal, protein, carbs, fat, goal}]
  const sys = `You are a nutrition coach. Use Australian English.`;
  const user = `Analyse the weight trend and calories vs goal. Identify adherence, suggest macro tweaks and weekly targets. Data: ${JSON.stringify(entries).slice(0,5000)}`;
  const r = await fetch('https://api.openai.com/v1/chat/completions',{
    method:'POST',
    headers:{'Content-Type':'application/json','Authorization':`Bearer ${process.env.OPENAI_API_KEY}`},
    body: JSON.stringify({
      model: 'gpt-4o-mini',
      messages: [{role:'system', content: sys},{role:'user', content:user}],
      temperature: 0.3
    })
  });
  const data = await r.json();
  res.status(200).json({ advice: data.choices?.[0]?.message?.content || 'No advice' });
}
